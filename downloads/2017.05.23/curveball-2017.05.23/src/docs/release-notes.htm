BBN Curveball release notes - version 2017.05.23

2017.05.23

This release is snapshot of a subset of the development tree of the BBN Curveball implementation of Decoy Routing, Quilting, and Remora. This software is a work in progress, and is under active development. This development version is known to have bugs, shortcomings, and weaknesses, and is only intended for experimental use.

Unless required by applicable law or agreed to in writing, this software is distributed on an "AS IS" BASIS, WITHOUT WARRANTIES OR CONDITIONS OF ANY KIND, either express or implied. See the license on each file or directory for the specific language governing permissions and limitations under the License.

This release contains the source code, build scripts, and instructions required to build the BBN Curveball client and servers for Ubuntu 12.04LTS and the BBN Curveball client for Windows 7. It also includes the source code and build scripts for building the client on MacOS X 10.8, but not instructions.

This release does not include all of the source code or instructions for building on Android.
